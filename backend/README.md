# Express Backend

A simple backend server built using **Node.js** and **Express**.

---

## Requirements

* Node.js
* npm

---

## Setup

Install dependencies:

```bash
npm install
```

---

## Run the Server

### Development mode

```bash
npm run dev
```

### Production mode

```bash
npm start
```

Server runs at:

```
http://localhost:3000
```

---

## Scripts Used

```json
"start": "node server.js",
"dev": "nodemon server.js"
```
